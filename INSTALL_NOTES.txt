Karman Kar Shows & Events — v10 Sponsorship Toggle Upgrade
===========================================================

Purpose
-------
Adds a per-show Allow Sponsorships switch so public sponsor links/forms can be hidden for events that do not use sponsors.

Files included
--------------
app.py
database.py
sponsorship_blueprint.py
templates/admin_shows.html
templates/base.html
templates/home.html
templates/show.html
templates/sponsorship_closed.html
templates/sponsorship.html
INSTALL_NOTES.txt

What changed
------------
1. Adds shows.allow_sponsorships with default ON for existing shows.
2. Adds Allow Sponsorships checkbox on Manage Shows create/update forms.
3. Hides public Sponsor links on:
   - top navigation
   - home page
   - show details page
4. Blocks direct /sponsorship/<show_slug> access when sponsorships are disabled.
5. Blocks public sponsorship form submits if sponsorships are disabled.
6. Adds a friendly sponsorship_closed.html page.

Install
-------
Copy these files into C:\karman-kar-shows and replace existing files when prompted.

Pre-deploy checks
-----------------
python -m py_compile app.py database.py sponsorship_blueprint.py
findstr /n /c:"allow_sponsorships" app.py database.py sponsorship_blueprint.py templates\admin_shows.html
findstr /n /c:"sponsorship_allowed" app.py templates\base.html templates\home.html templates\show.html
findstr /n /c:"sponsorship_closed.html" app.py sponsorship_blueprint.py

Git
---
git status --short
git add -A
git commit -m "Add per-show sponsorship toggle"
git push origin main

After deploy
------------
1. Open /admin/shows.
2. Edit a show and uncheck Allow Sponsorships.
3. Confirm Sponsor link disappears from top nav/home/show page.
4. Visit /sponsorship/<show-slug> directly and confirm the sponsorship unavailable page appears.
5. Re-check Allow Sponsorships and confirm sponsor links return.
