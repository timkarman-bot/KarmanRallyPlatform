# Paper Ballot Workflow

Version: 0.9.2-beta

Paper ballots are generated from the show's active judging/voting classes. Staff can enter paper ballots manually or import a CSV template generated by the platform.

Rules:
- One paper ballot may submit one 1st, one 2nd, and one 3rd choice per class.
- Car numbers must belong to the same class being voted on.
- Paper votes are inserted into the existing vote tally using categories like `Class Code - 1st`, `Class Code - 2nd`, and `Class Code - 3rd`.
- Current leaderboards and exports continue to work because paper votes also write to the `votes` table.

Admin pages:
- `/admin/paper-ballots` manual entry
- `/admin/paper-ballots/print` printable blank ballot
- `/admin/paper-ballots/template.csv` CSV template
- `/admin/paper-ballots/import` CSV import
